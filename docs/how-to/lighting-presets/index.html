GMK87 Lighting Presets and Effects

The GMK87 keyboard comes with 8 built-in lighting presets, 19 underglow effects, 5 LED modes, and 9 LED colors. This guide covers everything you need to configure your keyboard lighting exactly how you want it.

For full CLI reference and advanced configuration options, see the Configure Lighting guide.

Built-in Presets

Apply any preset instantly with a single command. Each preset configures underglow effects, LED color, brightness, and speed for a complete look.

gaming

Aggressive

Aggressive red theme with fast breathing effects. Full brightness, high saturation, and a fixed red LED. Built for intense gaming sessions where you want your setup to match the energy.

npm run preset gaming

work

Focused

Subtle white backlight for focused work. Uses a steady full-color underglow at moderate brightness so your keys are visible without distraction. The LED stays on a calm white.

npm run preset work

movie

Ambient

Purple wave effect with moderate brightness for a cinematic ambient look. Slow, flowing wave animation that adds atmosphere without pulling focus from the screen.

npm run preset movie

chill

Relaxed

Calm cyan breathing effect for relaxed typing. Slow speed, low brightness, and a soothing teal LED. Perfect for late-night browsing or casual use.

npm run preset chill

party

Flashy

Fast rainbow effects for maximum visual impact. Rainbow waterfall at full speed and brightness with a pulsing rainbow LED. The most eye-catching preset available.

npm run preset party

minimal

Clean

All lighting effects disabled for minimal distraction. Underglow off, LED color off, zero brightness. For users who want a clean, no-glow keyboard.

npm run preset minimal

off

None

Turns off all lighting completely. Both underglow and LED are disabled. Use this when you want absolute zero light output from your keyboard.

npm run preset off

default

Factory

Restores the keyboard to its default factory lighting configuration. Horizontal dimming wave with moderate brightness, rainbow mode enabled, and a blinking red LED.

npm run preset default

Underglow Effects

The GMK87 supports 19 underglow effects, each identified by an index number. Use the --effect flag to set one by name or number.

Index Effect Name Description
0offDisables underglow completely
1horizontal-dimming-waveHorizontal wave that dims across the keys
2horizontal-pulse-waveHorizontal wave with a pulsing intensity
3waterfallColor cascades downward like a waterfall
4full-cycling-colorsAll keys cycle through colors simultaneously
5breathingSmooth fade in and out like breathing
6full-one-colorSolid single color across all keys
7glow-pressed-keyLights up only the key you press
8glow-spreadingLight radiates outward from the pressed key
9glow-rowLights up the entire row of the pressed key
10random-patternRandom keys light up in random colors
11rainbow-cycleRainbow gradient cycles across the keyboard
12rainbow-waterfallRainbow colors cascade downward
13wave-from-centerLight wave radiates outward from the center
14circling-jkLight circles around the J and K keys
15rainingRandom keys light up and fade like rain drops
16wave-left-rightWave sweeps from left to right across keys
17slow-saturation-cycleSlowly cycles through saturation levels
18slow-rainbow-from-centerSlow rainbow that expands from the center
Example: Set the underglow to rainbow cycle at full brightness and fast speed:
node src/configureLights.js --effect rainbow-cycle --brightness 9 --speed 0

LED Modes

The big LED (screen area indicator) supports 5 display modes. Set it with the --led-mode flag.

Mode Name Description
0Blinking One ColorLED blinks in a single color at a steady interval
1Pulse RainbowLED pulses through rainbow colors smoothly
2Blinking One Color (Alt)Alternate blinking pattern in a single color
3Fixed ColorStays on a single solid color with no animation
4Fixed Color (Alt)Alternate fixed color display mode
Example: Set the LED to a fixed blue:
node src/configureLights.js --led-mode 3 --led-color blue

Custom Combinations

You can mix and match any underglow effect, LED mode, color, brightness, and speed to create your own setup. Pass multiple flags in a single command.

Available parameters

Flag Range Description
--effectname or 0-18Underglow effect
--brightness0-9Underglow brightness (0=off, 9=max)
--speed0-9Effect speed (0=fast, 9=slow)
--orientation0 or 1Direction (0=left-to-right, 1=right-to-left)
--rainbowtrue/falseEnable rainbow color cycling
--red --green --blue0-255Custom hue RGB values
--led-mode0-4LED display mode
--led-colorname or 0-8LED color preset
--led-saturation0-9LED saturation level
--led-rainbowtrue/falseLED rainbow mode

Examples

Cyan breathing at medium brightness with a fixed teal LED:

node src/configureLights.js --effect breathing --red 0 --green 255 --blue 255 --brightness 5 --speed 6 --led-mode 3 --led-color teal

Reactive key glow with rainbow enabled and a pulsing LED:

node src/configureLights.js --effect glow-pressed-key --rainbow true --brightness 7 --led-mode 1

Slow green rain effect (Matrix style) with a fixed green LED:

node src/configureLights.js --effect raining --red 0 --green 255 --blue 0 --brightness 7 --speed 2 --led-mode 3 --led-color green
Tip: You can also create your own preset file. Copy presets.json from the project root, modify it, and load it with node src/loadPreset.js mypreset --file ./my-presets.json.
